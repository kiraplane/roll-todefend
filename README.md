# Roll to Defend Wiki

Independent guide site for Roll to Defend at `https://www.roll-todefend.wiki`.

The site is built on the existing Next.js / MkSaaS stack and focuses on:

- Roll to Defend Roblox code status and safe verification notes
- Units, tier-list watch criteria, zones, luck boosts, and offline income
- A roll session planner for comparing rolls, upgrades, zones, and waiting
- Official Roblox page, D:/Drive group, Discord/Trello status, and update-signal guidance
- Script, fake link, and copied-board safety for Roblox players

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
```

## Content Rules

- Do not publish unverified redeem codes as guaranteed active.
- Treat tier-list demand as watch material until Roll to Defend unit data is reliable.
- Prefer the official Roblox page, Roblox API facts, official group/community updates, current search results, and current creator videos.
- Keep the site clearly labeled as an independent guide hub.
- Do not provide scripts, executors, modified clients, APK mirrors, or fake reward instructions.
